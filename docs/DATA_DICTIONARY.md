# Data Dictionary

| Column Name   | Description                                            |
| ------------- | ------------------------------------------------------ |
| Sale_ID       | Unique identifier for each sales transaction           |
| Doctor_ID     | Unique identifier for the doctor                       |
| Hospital_ID   | Unique identifier for the hospital                     |
| Product_ID    | Unique identifier for the pharmaceutical product       |
| Region_ID     | Unique identifier for the sales region                 |
| Time_ID       | Links the sale to the time dimension                   |
| Quantity_Sold | Number of units sold                                   |
| Revenue       | Total revenue generated from the sale                  |
| Unit_Price    | Price per unit of the product                          |
| Discount      | Discount applied to the sale                           |
| Profit        | Profit earned from the transaction                     |
| Sales_Channel | Sales channel used (Online, Retail, Distributor, etc.) |
| Payment_Type  | Payment method used                                    |
| Customer_Type | Category of customer (New, Existing, etc.)             |
